Projects

Things I've Built

📄
ResultFlow
Automates VTU result analysis end-to-end

VTU result PDFs are notoriously messy — inconsistent formatting, multi-page layouts, and no structured export. ResultFlow parses these raw PDFs with surgical precision, extracts student marks and metadata, and generates clean analytical reports. What used to take hours of manual entry now takes seconds, benefiting hundreds of students and faculty alike.

Flask pdfplumber pandas Python
📧
Email Tracer
Traces email origin & routing in real-time

Phishing and spoofed emails are a growing threat. Email Tracer is a Flutter mobile app that parses raw email headers and visually maps the mail routing path — revealing sender origin, relay servers, and potential spoofing attempts. Built for security-conscious professionals and developers who want clarity on where an email actually came from.

Flutter Dart Email Headers API
🛡️
Vulnerability Testing Tool
Detects SQL injection risks & open ports

Security vulnerabilities cost organizations millions. This Flutter app provides developers and security researchers with a lightweight mobile toolkit for SQL injection testing and port scanning. By enabling vulnerability detection directly from a smartphone, the tool democratizes security testing and helps catch critical flaws before they become breaches.

Flutter Dart Network APIs
